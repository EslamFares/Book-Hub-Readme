## Features 🌟

- 🌍 Clean architecture
- 🚀 Firebase auth || Firebase notification.
- 💡 Localization || Change theme.
- 🔐 Pagination || Deep link.
- 🧠 Dependency injection || Refresh. 
- 🍯 Hive local storage || Dio Remote source.
